**Lexis**, Türkçe dil bilgisine ve kelime haznesine güvenenler için özel olarak geliştirilmiş rekabetçi bir kelime tahmin oyunudur. Klasik kelime tahmin mantığını yepyeni oyun modlarıyla harmanlar ve arkadaşlarınızla gerçek zamanlı online rekabet etme şansı sunar.

## ✨ Özellikler

* **🌐 Online Eşleşme:** Gerçek oyuncularla anlık olarak yarışın. Aynı kelimeyi kim daha az hamlede, daha kısa sürede bulacak?
* **🏆 Puanlı Seri Modu:** Arka arkaya kaç kelime bilebileceksin? Skorunu liderlik tablosunda herkese göster!
* **🎁 Günlük Mod:** Her gün herkes için özel olarak seçilmiş aynı kelimeyi bil, istatistiklerini arkadaşlarınla paylaş.
* **🃏 Joker Sistemi:** Tıkandığın yerde ünlü harf, ünsüz harf açtırabilir ya da ekstra tahmin hakkı kullanabilirsin.
* **🔄 Uygulama İçi Otomatik Güncelleme:** Oyun içi GitHub entegrasyonu sayesinde her zaman en güncel sürümde kalırsın!
* **🌓 Karanlık ve Aydınlık Mod:** Göz yormayan şık ve modern (Glassmorphism) tasarım.

## 🛠️ Teknolojiler

Bu proje güncel ve modern mimari yaklaşımlarla geliştirilmiştir:
- **Framework:** Flutter (Cross-platform)
- **State Management:** Riverpod (ConsumerStatefulWidget)
- **Backend & Database:** Firebase Auth & Cloud Firestore
- **CI/CD:** GitHub Actions (Otomatik APK derleme ve Yayınlama)

## 💻 Geliştiriciler İçin (Kurulum)

Projeyi kendi bilgisayarınızda derlemek isterseniz:

1. Depoyu klonlayın:
```bash
git clone https://github.com/mfbilgin/lexis-game.git
```
2. Bağımlılıkları yükleyin:
```bash
flutter pub get
```
3. Uygulamayı çalıştırın:
```bash
flutter run
```
